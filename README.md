# Blind Wave

**Blind Wave** is a sonar-based exploration game where you navigate a pitch-black maze using sound waves.

## How to Play (Laptop/Desktop)

**Objective:** Navigate the dark maze using sonar pulses. Find the **Exit Zone** (faint white box) to advance. Avoid hitting walls!

### Controls
*   **Move:** Use `W`, `A`, `S`, `D` keys OR `Arrow Keys`.
*   **Send Sonar Pulse:** Press `Spacebar` OR `Left Mouse Click`.
*   **Switch Frequency:**
    *   Press `1`: **Red Frequency** (Reveals Red walls)
    *   Press `2`: **Green Frequency** (Reveals Green walls)
    *   Press `3`: **Blue Frequency** (Reveals Blue walls)

### Gameplay Tips
*   **Red walls** are invisible unless you ping them with **Red Frequency**.
*   **Green walls** are invisible unless you ping them with **Green Frequency**.
*   **Blue walls** are invisible unless you ping them with **Blue Frequency**.
*   You must switch frequencies rapidly to see the full picture of the maze!

## How to Run
1.  Open your terminal in this folder.
2.  Run `npm install` (to install dependencies).
3.  Run `npm run dev`.
4.  Open the local URL shown in the terminal (usually `http://localhost:5173`) in your web browser.
